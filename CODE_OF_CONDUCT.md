# Code of Conduct

## Our Standard

This project welcomes contributors of all backgrounds.
All interactions in issues, pull requests, discussions, and reviews are expected to be respectful, professional, and constructive.

Unacceptable behavior includes harassment, personal attacks, discrimination, intimidation, and deliberate disruption of collaboration.

## Enforcement

Maintainers may remove comments, close discussions, reject contributions, or block participation when behavior violates this code.

## Reporting

Report conduct concerns privately to the maintainers through GitHub security advisories or by opening a confidential maintainer contact request in this repository.

When reporting, include links/screenshots and a short timeline so maintainers can act quickly.

## Scope

This code applies to all repository spaces and project-managed communication channels.
